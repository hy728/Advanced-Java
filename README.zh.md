# 项目介绍
## 背景
在当代数字时代，酒店行业正经历着由技术进步驱动的重大变革。对高效、用户友好且功能丰富的酒店管理系统的需求比以往任何时候都更加关键。兰登酒店应用程序（LandonHotelApp）就是在这样的背景下开发的。随着酒店运营的日益复杂，如客房管理、预订处理以及为不同客户群体提供多语言服务等，对综合性解决方案的需求十分迫切。该应用程序旨在弥合传统酒店管理方法与数字时代的现代需求之间的差距，提升酒店的运营效率以及客人的入住体验。（作者：[你的名字]）

## 目标
1. 提高运营效率：实现各种酒店运营的自动化和流程优化，包括客房管理、预订处理和数据收集。通过减少人工任务，该应用程序旨在最大限度地减少错误，并为酒店员工节省时间，使他们能够专注于提供高质量的客户服务。
2. 改善客户体验：为客人提供一个直观便捷的平台，用于搜索可用客房、查看客房详情、进行预订以及访问多语言欢迎信息。该应用程序致力于满足来自不同文化背景的客人的多样化需求，提供无缝且个性化的体验。
3. 数据驱动的决策制定：收集并分析与客房入住率、预订模式和客户偏好相关的数据。这些数据将用于支持酒店的战略决策，如定价策略、营销活动和资源分配。（作者：[你的名字]）

## 功能概述
1. 客房管理：该系统为酒店管理员提供一套全面的工具来管理客房信息。他们可以添加、编辑或删除客房详细信息，包括房间号、房间类型、价格和可用状态。根据维护计划或其他因素，管理员可以实时将客房标记为可用或不可用。
2. 预订处理：客人可以通过指定入住和退房日期轻松搜索可用客房。该应用程序将显示所有可用客房的列表以及相应的价格和房间类型。然后，客人可以直接通过平台进行预订，酒店员工可以管理这些预订，包括确认、修改或取消预订。
3. 多语言支持：为了满足国际客人的需求，该应用程序提供多语言支持。客人可以选择他们喜欢的语言来查看欢迎信息并使用该应用程序。此功能确保来自不同文化背景的客人能够轻松理解并与应用程序进行交互。
4. 单元测试和端到端测试：该应用程序内置了对单元测试和端到端测试的支持。开发人员可以通过 [Karma](https://karma - runner.github.io) 使用 ng test 运行单元测试，并在添加必要的测试包后使用 ng e2e 进行端到端测试。（作者：[李悠扬]）

## 主要功能和教程
1. 前提条件
Java：确保系统已安装 Java 17。
Node.js 和 npm：运行 Angular 前端需要。可从 Node.js 官网（https://nodejs.org/）下载。
Maven：用于构建 Java 项目。可从 Apache Maven 官网（https://maven.apache.org/）下载。
Git：用于克隆项目。从 Git 官网（https://git-scm.com/）安装。
2. 克隆项目
打开 Git Bash 或首选终端：导航至想克隆项目的目录。
克隆仓库：运行以下命令：
git clone <仓库地址>

3. 构建项目
3.1 Java 后端
导航至项目根目录：在项目根目录打开终端。
使用 Maven 构建 Java 项目：运行以下命令：
在 Maven 工具窗口的工具栏中：
点击下载源代码和文档图标（该图标通常是向下箭头或包含 ↓ 符号）。

或者通过右键菜单：
右键点击项目的根节点（或特定模块），选择 Maven > 下载源代码和文档。
mvn clean install

3.2 Angular 前端 --by pangyadan
导航至前端目录：打开新终端，导航至 src/main/UI 目录。
安装依赖项：运行以下命令：
npm install

4. 运行应用程序
4.1 启动 Java 后端
导航至项目根目录：在项目根目录打开终端。
运行 Spring Boot 应用程序：运行以下命令：
mvn spring-boot:run

4.2 启动 Angular 前端
导航至前端目录：打开新终端，导航至 src/main/UI 目录。
ng serve
注意：Angular 应用程序将在 http://localhost:4200 上可用。


5. 使用酒店预订系统
5.1 预订房间
在浏览器中打开应用程序：在 Web 浏览器中访问 http://localhost:4200。
输入入住和退房日期：在 "预订房间" 表单中填写入住和退房日期。
点击 "提交" 按钮：输入日期后，点击 "提交" 按钮。
注意：应用程序将根据所选日期显示可用房间。

5.2 预订房间 
选择房间：在 "可用房间" 部分，找到想要预订的房间。
点击 "预订" 按钮：点击所选房间旁边的 "预订" 按钮。
注意：这将为所选房间创建预订。


6. 多线程语言翻译和货币兑换
语言翻译：应用程序支持多线程语言翻译。应用程序启动时，您可以看到不同语言的欢迎消息。
注意：英语和法语欢迎消息使用多线程编程显示。
货币兑换：房间价格以不同货币（美元、加元、欧元）显示。
注意：这为来自不同国家的用户提供了一种方便的查看价格方式。

7. 停止应用程序
停止 Spring Boot 应用程序：在运行 Spring Boot 应用程序的终端中按 Ctrl + C。
停止 Angular 开发服务器：在运行 Angular 开发服务器的终端中按 Ctrl + C。
